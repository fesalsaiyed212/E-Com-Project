# Fesal Saiyed — Mini E‑commerce (GitHub-ready)

Full working mini e‑commerce demo: products, cart, orders, CSV export.

## Run locally
```bash
npm install
npm start
# open http://localhost:3000
```

## Push to GitHub
```bash
git init
git add .
git commit -m "Initial commit: Fesal Saiyed e-commerce"
git branch -M main
git remote add origin https://github.com/<your-username>/<your-repo>.git
git push -u origin main
```
